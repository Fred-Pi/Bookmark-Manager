# Screenshots Guide

## How to Take Screenshots

### 1. Main Dashboard (dashboard.png)
- Open https://bookmark-manager-xi-ochre.vercel.app
- Log in and add a few bookmarks with tags
- Show the full interface with:
  - Search bar
  - Sort dropdown
  - Import/Export buttons
  - At least 6-8 bookmarks displayed
  - Tags visible
- Take a full-page screenshot

### 2. Add Bookmark Form (add-bookmark.png)
- Focus on the left sidebar with the "Add Bookmark" form
- Fill in a sample bookmark
- Show the form fields clearly
- Capture the form area

### 3. Toast Notification (toast-notification.png)
- Add or delete a bookmark to trigger a toast
- Take screenshot while toast is visible in top-right
- Show the green success toast if possible

### 4. Browser Extension (extension.png)
- Install the extension from the `extension/` folder
- Open the extension popup
- Show it filled out on a real webpage
- Capture the extension popup

## Screenshot Tools

**Mac:**
- Cmd + Shift + 4 (select area)
- Cmd + Shift + 3 (full screen)

**Windows:**
- Windows + Shift + S (Snipping Tool)
- Or use browser DevTools screenshot feature

**Browser DevTools (Recommended for web app):**
1. Open DevTools (F12)
2. Cmd/Ctrl + Shift + P
3. Type "screenshot"
4. Choose "Capture full size screenshot" or "Capture screenshot"

## File Naming
Save screenshots in this folder as:
- `dashboard.png`
- `add-bookmark.png`
- `toast-notification.png`
- `extension.png`

After adding screenshots, commit them to git!
